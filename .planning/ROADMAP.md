# Project Roadmap

**Current Milestone**: Phase 1

### Phase 1: Environment & Tooling Baseline
- Setup Next.js with Tailwind and Framer Motion.
- Setup FastAPI structure.
- Initialize PostgreSQL Database schema using Vercel/Render or Supabase.

### Phase 2: Design System & Core UI Shell
- Implement custom Tailwind themes for premium startup look (glassmorphism, gradients).
- Build the 3D-inspired landing page with central orb animation.
- Implement central navigation and layout shell.

### Phase 3: AI & Voice Backend Engine
- Integrate OpenAI for Whisper (Speech-to-Text).
- Create Intent Detection endpoint using LLM.
- Route intents to specific JSON data shapes.

### Phase 4: Financial Modules & Logic
- Build frontend FD Calculator, Compare Banks, and Goal Planning modules.
- Link frontend visual states with the backend logic.
- Ensure hybrid input (text forms + voice context updates) works seamlessly.

### Phase 5: Workflow Integration & Polish
- Ensure every flow suggests the Next Action.
- Verify user session memory is being persisted.
- End-to-End polish (transitions, animations, loading states).
